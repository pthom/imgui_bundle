# tests/ — Test suite

**Quick start:**
```bash
just test_pytest    # Run pytest
just test_mypy      # Type-check Python bindings
```

**Contents:**
- `lg_imgui_bundle_test.py` — Main test file (litgen binding generation tests)
- `tests_python_gui/` — GUI tests (require a display): widget rendering, async integration
